функция для создания табов на странице

=== Вызов функции
createTab(ClassTab, ClassAnsesterTab, ClassContentTab)

Аргумент ClassTab -- класс пункта меню в табуляторе, синтаксис '.classNameTab';
Аргумент ClassAnsesterTab -- класс родительского элемента пунктов меню, синтаксис '.className';
Аргумент ClassContentTab -- класс контента таба синтаксис '.className';

=== CSS файл
класс .fade анимированный переход от файла к файлу
класс .show класс для отображения таба
класс .hide класс для скрытия таба
